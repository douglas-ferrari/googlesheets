Fundamentos de Fórmulas e Funções no Google Planilhas

1. Conceitos: Fórmulas vs. Funções

No Google Planilhas, é essencial diferenciar Fórmulas de Funções:

  • Fórmula: É a expressão completa que você insere em uma célula para realizar um cálculo. Ela sempre começa com o sinal de igual (=). Uma fórmula pode conter operadores, referências de células, valores fixos (constantes) e funções.
    Exemplo: =A1+B1 ou =SOMA(A1:A5) / 10
  • Função: É uma operação predefinida e nomeada que realiza um cálculo específico, simplificando fórmulas complexas. Funções recebem "argumentos" (dados para processar) dentro de parênteses.
    Exemplo: SOMA, MÉDIA, SE, PROCV

Em resumo: você escreve uma fórmula que pode (ou não) usar funções.

2. A Sintaxe de uma Fórmula

Toda fórmula inicia com = e é seguida pelos elementos de cálculo. Os argumentos dentro das funções são separados por ponto e vírgula (;) na configuração regional do Brasil.

Exemplo 1: Fórmula com Operadores

Para somar o valor da célula A1 com B1:

=A1+B1

Exemplo 2: Fórmula com Função

Para somar todos os valores no intervalo (range) de A1 até A5:

=SOMA(A1:A5)

Aqui, A1:A5 é o argumento que define o intervalo de células para a função SOMA.

3. Operadores Aritméticos e Ordem de Precedência

Operadores são símbolos usados para executar operações matemáticas básicas:

  • Adição: + (ex: =A1+B1)
  • Subtração: - (ex: =A1-B1)
  • Multiplicação: * (ex: =A1*B1)
  • Divisão: / (ex: =A1/B1)
  • Exponenciação (Potência): ^ (ex: =A1^2)
Importante: Ordem de Precedência

O Google Planilhas segue a ordem matemática padrão (PEMDAS/BODMAS) para calcular:

  1. Parênteses ( )
  2. Expoentes ^
  3. Multiplicação * e Divisão /
  4. Adição + e Subtração -

Use parênteses para forçar a ordem desejada. =(A1+A2)*10 é diferente de =A1+A2*10.

4. Funções Essenciais

Funções automatizam cálculos comuns. Aqui estão algumas das mais usadas:

Funções Matemáticas e Estatísticas

  • =SOMA(intervalo): Soma todos os números em um intervalo de células.
    Ex: =SOMA(A1:A10)
  • =MÉDIA(intervalo): Calcula a média aritmética dos números no intervalo.
    Ex: =MÉDIA(A1:A10)
  • =MÁXIMO(intervalo): Retorna o maior valor numérico no intervalo.
    Ex: =MÁXIMO(A1:A10)
  • =MÍNIMO(intervalo): Retorna o menor valor numérico no intervalo.
    Ex: =MÍNIMO(A1:A10)
  • =CONT.NÚM(intervalo): Conta quantas células contêm números no intervalo.
    Ex: =CONT.NÚM(A1:A10)

5. Função Condicional SE (IF)

A função SE é uma das mais poderosas. Ela permite que a planilha tome decisões, retornando um valor se uma condição for verdadeira e outro valor se ela for falsa.

Estrutura: =SE(teste_logico; valor_se_verdadeiro; valor_se_falso)

  • teste_logico: A condição que você quer verificar (ex: A1 > 10, B2 = "Aprovado").
  • valor_se_verdadeiro: O que a célula deve exibir se o teste for VERDADEIRO.
  • valor_se_falso: O que a célula deve exibir se o teste for FALSO.

Exemplo: Verificar se um aluno foi aprovado (nota em A1 maior ou igual a 7):

=SE(A1>=7; "Aprovado"; "Reprovado")

6. Fundamento Essencial: Tipos de Referência de Célula

Quando você copia e cola uma fórmula, as referências de célula (como A1) podem mudar. Controlar isso é vital.

Referência Relativa (o padrão)

=A1: Quando você copia esta fórmula para baixo, ela se ajusta (A2, A3...). Quando copia para o lado, ela também se ajusta (B1, C1...).

Referência Absoluta (fixa/travada)

=$A$1: O símbolo $ (cifrão) "trava" a coluna (A) e a linha (1). Não importa para onde você copie a fórmula, ela sempre apontará para $A$1.
Uso: Ideal para referenciar uma taxa de juros, um imposto ou uma constante que está em uma célula fixa.

Referência Mista

=$A1 (Coluna absoluta, linha relativa): A coluna A fica travada, mas a linha 1 se ajusta ao ser copiada para baixo.

=A$1 (Coluna relativa, linha absoluta): A coluna A se ajusta ao ser copiada para o lado, mas a linha 1 fica travada.

Dica: Use a tecla F4 no seu teclado com o cursor sobre uma referência (como A1) dentro da fórmula para alternar rapidamente entre os tipos de referência (A1 -> $A$1 -> A$1 -> $A1).

7. Boas Práticas e Erros Comuns

  • Separador de Argumentos: No Brasil, use ponto e vírgula (;) para separar argumentos (ex: =SE(A1>10; "Sim"; "Não")). Em inglês (EUA), usa-se vírgula (,).
  • Separador Decimal: No Brasil, use vírgula (,) para decimais (ex: 10,50).
  • Evite "Hardcoding": Não insira valores fixos (como taxas) diretamente nas fórmulas (ex: =A1 * 0,05). É melhor colocar o valor 0,05 em uma célula (ex: C1) e usar uma referência absoluta: =A1 * $C$1. Assim, se a taxa mudar, você só altera a célula C1.
  • Erros Comuns:
    • #DIV/0!: Tentativa de divisão por zero.
    • #REF!: A fórmula usa uma referência de célula que não existe mais (ex: você excluiu a coluna).
    • #NOME? (ou #NAME?): O Planilhas não reconhece o nome da função (ex: digitou =SOMA errado).
    • #VALOR!: A fórmula recebeu um tipo de dado inesperado (ex: tentar somar texto com número).
